## Wiki organization methodologies, taxonomies, structures

### Organizing using the Library of Congress subject headings

- [Search Obsidian plugins for "congress"](https://obsidian.md/plugins?search=congress)
- [kometenstaub/linked-data-vocabularies](https://github.com/kometenstaub/linked-data-vocabularies) — uses Library of Congress taxonomy/folksonomy.
- Had to download the MADS/RDF JSONLD file from [id.loc.gov/download](https://id.loc.gov/download/).
- Set the path in the linked-data-vocabularies tool.
- Then query: `Ctrl + P` > LCSH > type in category.
- The Breadcrumbs plugin can help facilitate use of this plugin as well.
- Not generating JSON files: ["Linked data helper" plugin by kometenstaub — bug graveyard](https://forum.obsidian.md/t/linked-data-helper-plugin-by-kometenstaub/28875).

- **id.loc.gov API — Linked Data and Subject Headings:**
  - Each controlled vocabulary list has a resolvable URI, as does each data value within it. Ontologies for several major data element sets are also available.
  - Links:
    - [Additional APIs and Data Services — LoC.gov](https://www.loc.gov/apis/additional-apis/)
    - [Linked Data Service — LoC.gov](https://www.loc.gov/apis/additional-apis/linked-data-service/)
    - [Library of Congress Subject Headings (id.loc.gov)](https://id.loc.gov/authorities/subjects.html)
    - [Data for Exploration — LoC](https://data.labs.loc.gov/apis/)
    - [Working Within Limits — JSON/YAML — LoC](https://www.loc.gov/apis/json-and-yaml/working-within-limits/#rate-limits)
    - [Integrating Wikidata at the Library of Congress — The Signal](https://blogs.loc.gov/thesignal/2019/05/integrating-wikidata-at-the-library-of-congress/)
  - Using the "Linked Data Service" API for subject headings:
    - [Technical Center — Downloads (id.loc.gov)](https://id.loc.gov/techcenter/)
    - [Technical Center — Searching (id.loc.gov)](https://id.loc.gov/techcenter/searching.html)

- Related but not the desired API — LoC.gov API:
  - [JSON/YAML for LoC.gov](https://www.loc.gov/apis/json-and-yaml/) — not for the "linked data" / subject headings.
  - [LibraryOfCongress/data-exploration](https://github.com/LibraryOfCongress/data-exploration)
  - [Tutorials for Data Exploration (libraryofcongress.github.io)](https://libraryofcongress.github.io/data-exploration/intro.html)
